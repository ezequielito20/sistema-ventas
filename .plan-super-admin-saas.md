# Plan: Panel Super Admin + SaaS

> **Stack:** Laravel 13 + Livewire 3 + Alpine.js + Tailwind CSS + PostgreSQL + Spatie Permission  
> **Patrón:** Livewire Component → Blade View → Service class  
> **UI System:** Componentes `x-ui.*`, `ui-panel`, `ui-btn`, `ui-table`, `ui-badge`, `ui-icon-action`  
> **Notificaciones:** Toast via `$this->toast()` en Livewire

## FASE 4 — Servicios (Business Logic)

### 4.1 `SubscriptionService`
**Archivo:** `app/Services/SubscriptionService.php`

**Métodos:**
- `createForCompany(Company $company, Plan $plan, int $billingDay = 1): Subscription`
  - Crea suscripción con 30 días de trial o activa según config
  - Setea `next_billing_date` y `grace_period_end`
- `calculateAmount(Company $company, Plan $plan): float`
  - `base_price + (usuarios_extra * price_per_user) + (transacciones_extra * price_per_transaction) - descuentos`
- `renewSubscription(Subscription $subscription): void`
  - Actualiza periodo, setea nueva `next_billing_date`
- `generateMonthlyPayment(Subscription $subscription): SubscriptionPayment`
  - Crea el pago del período, calcula monto, setea `due_date` y estado `pending`
- `activateGracePeriod(Subscription $subscription): void`
  - Pasa de `active` a `grace_period`, setea `grace_period_end = due_date + 4 days`
- `suspend(Subscription $subscription, string $reason = null): void`
  - Pasa a `suspended`, setea `cancelled_at`
- `activate(Subscription $subscription): void` → reactivar
- `cancelSubscription(Subscription $subscription, string $reason): void`
- `changePlan(Subscription $subscription, Plan $newPlan): void`

### 4.2 `PaymentService`
**Archivo:** `app/Services/PaymentService.php`

**Métodos:**
- `markAsPaid(SubscriptionPayment $payment, int $processedBy, ?UploadedFile $receipt = null): void`
  - Actualiza estado, guarda comprobante con `ImageUrlService`, setea `paid_at` y `processed_by`
- `uploadReceipt(SubscriptionPayment $payment, UploadedFile $file): string`
  - Guarda en `subscription_receipts/{company_id}/`
- `getReceiptUrl(?string $path): ?string` → via `ImageUrlService`
- `cancelPayment(SubscriptionPayment $payment): void`
- `getPendingPayments(): Builder` → todos los pendientes/vencidos
- `getPaymentsByCompany(int $companyId): Collection`

### 4.3 `UsageCollectorService`
**Archivo:** `app/Services/UsageCollectorService.php`

**Métodos:**
- `collectForCompany(int $companyId): SubscriptionUsageLog`
  - Cuenta usuarios, ventas del mes, productos, clientes, total revenue
  - Guarda snapshot en `subscription_usage_logs`
- `collectForAllActiveCompanies(): void`
  - Itera todas las empresas activas y llama `collectForCompany`

### 4.4 `CompanyService` (extender existente o nuevo)
**Archivo:** `app/Services/CompanyService.php`

**Métodos:**
- `createFromSuperAdmin(array $data): Company`
  - Crea empresa con datos mínimos (name, nit, email)
  - Crea usuario admin con email/password provisto
  - Asigna plan Básico
  - Genera primera suscripción
- `statistics(): array` → stats globales para dashboard super admin
- `topByRevenue(int $limit = 10): Collection`
- `topBySales(int $limit = 10): Collection`
- `topByCustomers(int $limit = 10): Collection`
